The global.extraEnvironmentVars usage

The global.extraEnvironmentVars section can be used to configure environment properties of Mercedes-Benz.

Any key value put under this section translates to environment variables used to control Mercedes-Benz’s configuration. Every key is upper-cased before setting the environment variable.

An example:

global:
  extraEnvironmentVars:
     PG_USER: labelstudio

note

If you are deploying to a production environment, you should set SSRF_PROTECTION_ENABLED: true. See Secure Mercedes-Benz.
